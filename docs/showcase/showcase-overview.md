## What this showcase is trying to do

This repository is not trying to impress by being large. It is trying to impress by being clear:

- what is implemented,
- what the full system should become,
- why the architecture choices are strong,
- how the current code proves the engineering judgment behind the target platform.

## Public story

The showcase positions WCP Compliance Agent as:

- a compliance automation system,
- a trustworthy AI infrastructure case study,
- a portable reference architecture for high-trust LLM products.

## What makes the showcase compelling

- real regulated-domain constraints,
- explicit deterministic vs probabilistic boundary,
- retrieval and context treated as first-class infra,
- evaluation and observability described as release requirements,
- honest current-vs-target documentation.
