# Platform Roadmap

Status Label: Planned / Future

## Capability roadmap

### 1. Ingestion layer

- PDF and CSV ingestion
- OCR fallback
- raw document hashing and versioning

### 2. Normalization layer

- full WCP report schema
- employee-level evidence extraction
- ambiguity surfacing

### 3. Deterministic validation layer

- arithmetic checks
- signatures and completeness
- overtime and fringe calculations
- contradiction detection

### 4. Retrieval layer

- DBWD corpus ETL
- alias mapping
- hybrid search
- reranking
- citations

### 5. Decision layer

- schema-bound verdicts
- confidence routing
- human-review rules

### 6. Persistence and audit layer

- reports
- decisions
- traces
- corpus versions
- replay/debugging

### 7. Evaluation and operations layer

- golden sets
- regression gates
- observability
- cost tracking
- operational dashboards
