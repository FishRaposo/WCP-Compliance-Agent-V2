## Current bar

Current acceptance bar for the proof artifact:

- build passes,
- deterministic tests pass,
- orchestration test passes,
- public documentation stays honest about implementation scope.

## Target production bar

- verdict agreement >= 90 percent on labeled set,
- false-approve rate < 2 percent,
- citation validity >= 95 percent,
- end-to-end latency P95 < 20 seconds for standard documents,
- structured output integrity at effectively 100 percent,
- explicit escalation on low-confidence or conflicting evidence.

## What this document is for

This is the benchmark doc that keeps the future system from drifting into vague "good enough" language.
